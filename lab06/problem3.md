字符全排列

描述
给定一个由不同的小写字母组成的字符串，输出这个字符串的所有全排列。

输入
输入只有一行，是一个由不同的小写字母组成的字符串，已知字符串的长度在2到8之间。

输出
输出这个字符串的所有排列方式，每行一个排列。要求字母序比较小的排列在前面。字母序如下定义：

已知 S = s1s2...sk, T = t1t2...tk，则 S < T 等价于，存在 p (1 <= p <= k)，使得 s1 = t1, s2 = t2, ..., sp - 1 = tp - 1, sp < tp 成立。

输入样例 1
```
abc
```

输出样例 1
```
abc
acb
bac
bca
cab
cba
```
