# 删除链表中重复的节点

### 描述
在一个排序的链表中，存在重复的节点，请删除该链表中重复的节点，重复的节点都不保留。

### 输入
输入一个已排序的链表，链表节点的值之间用空格隔开，以 -1 作为结束标志（节点不包含 -1）。

数据范围：链表中节点 val 取值范围 [0, 100]。链表长度 [0, 100]。

### 输出
输出删除重复节点后的链表，如果删除后链表为空，则输出 NULL。

输入样例 1
```
1 2 3 4 3 4 5 -1
```

输出样例 1
```
1 2 5
```

输入样例 2
```
1 1 1 1 2 2 2 -1
```

输出样例 2
```
NULL
```
